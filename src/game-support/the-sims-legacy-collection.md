## Setup

- Install in Steam as normal.
- Run DLL winetricks: `vcrun2022`, `dsound`, `wmp9`, `wmp10` and `wmp11`.
- Run with argument `-monitor:X` to play on a different monitor, with `X` being the monitor number.
